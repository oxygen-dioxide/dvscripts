# dvlrc
由deepvocal文件制作lrc歌词，需要 python dvfile >=0.0.6

## 使用方法

第1步：双击运行dvlrc.py，将dv文件拖入窗口，回车，（如果有多条音轨）选择音轨

第2步：在弹出的记事本窗口中，用回车断句，保存，关闭记事本，在dv文件同一文件夹下生成lrc文件

## 说明
1. lrc文件的时间轴以dv文件第一小节起始位置为零点

2. 默认在windows下运行，调用系统记事本为文本编辑器。如果要更换文本编辑器或在其他系统运行，请自行修改texteditor函数（打开文本编辑器编辑输入的字符串，并按行返回编辑后的字符串列表）
